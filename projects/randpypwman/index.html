RandPyPwMan.

Version: 0.6.2

A simple and easy to use password generator and manager, made by me, Hayden Hildreth. Written in Python using the Tkinter face and SQLite3.

The password manager is complete with the hashing of master password, encryption of stored passwords and even the ability to search passwords by group.
