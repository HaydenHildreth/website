Windows update.

Published: Oct 15, 2025

My day today started with texts and emails about the software not working. And it wasn't. After troubleshooting, and researching, I found the root cause of the issue to be a Windows update.
Uninstalling the updates the machine had installed yesterday evening, resolved any issues the customers were having. So I just wanted to take this time to write an angry rant about it here, as I don't have anyone else to complain about this with. Just a day in the life of an IT personnel.
Thank you, Micro$oft!
